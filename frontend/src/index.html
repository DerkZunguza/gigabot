Telegram — @Gigabotmz_bot

Controla o bot remotamente pelo Telegram. Envia os comandos abaixo para @Gigabotmz_bot.

/status
Estado de todos os sistemas (WhatsApp, MQTT, Arduino)
/qr
Recebe o QR Code ou codigo de pareamento directamente no Telegram
/restart
Reinicia a conexao WhatsApp e gera novo QR
/pedidos
Resumo das vendas do dia (total, activados, pendentes, receita)
/chatid
Mostra o teu Chat ID para configurar o TELEGRAM_CHAT_ID
/ajuda
Lista todos os comandos disponiveis
